文件后缀名

sass有.sass和.scss两种后缀文件名。.sass是不使用大括号和分号。.scss是平常使用
css文件格式。常规使用.scss文件格式，以避免出现sass格式错误而出现的编译错误。

文件导入

sass的文件导入采用@import ‘文件名’的方式，在此您可以不添加后缀名。sass导入后会生成一个文件，若以@import ‘reset.css’的方式导入，择会以css方式导入，并不会生成一个文件。

变量

sass变量有常规变量采用$开头，后面紧跟变量名，而变量值和变量名之间添加冒号（:）分隔开，如果后面添加！important则为默认值。

默认变量

sass的默认变量仅需要在值后面加上 !default 即可。

sass的默认变量一般用来设置默认变量，然后根据需求来覆盖、

变量类型

普通变量

定义之后可以在任何位置使用。

特殊变量使用时需要 #{$variable} 使用

###嵌套
&表示本身

选择器嵌套

属性嵌套

###@at-root跳出嵌套

@at-root 和&一起可以在交换选择器嵌套

@at-root只能跳出选择器嵌套。不能跳出@medio或@support，需要跳出@media和@support时使用@at-root(without:media)。常用@media，@support兼容性有问题。

###mixin 混合用$mixin:定义。@include 引用混合。

继承 @extend

@extend 样式块引用。

%定义站位选择器

这种选择器的优势是，在你不调用的时候不会有css文件。这样避免了定义大量的基础样式，却又有很大一部分没有使用的情况
可以定义基础样式，当样式内容引用了基础样式的时候sass才会加载渲染

函数@function

主要使用颜色函数，加深和减淡。

###运算
sass具有运算的特性，可以对数值的Value（如：数字、颜色、变量等）进行加减乘除四则运算。运算符前后需要添加空格进行运算，不然会出错。

条件判断及循环

@if

@if可以单独使用，也可以结合@else多条件使用

三目运算

语法为：if($condition,@if_true,@if_false)。三个参数分别表示条件，条件为真的值，和条件为假的值。

$animal-list:puma,sea-slug,egret,salmander;

.#{$animal}-icon{
background-image:url(‘/images/#{animal}.png’);
}
}
//css style
.puma-icon{
background-image:url(‘/images/puma.png’);
}

}
.egret{
background-image:url(‘/images/egret.png’);
}
.salmander{
background-image:url(‘/images/salmander.png’);
}

JavaScript 数据类型

javascript中有基本类型null，undefined，Number，string，boolean，和引用类型对象Object；

JavaScript是松散类型，可以采用typeof来

Undefined类型

undefined类型只有一个值即undefined。在使用var定义变量但未对其初始化时，此时变量的值就为undefined。

包含未经定义值的变量和未定义的变量两者是不一样的。

上面可以看到当未被初始化的变量获取其数据类型时，此时是undefined。

Null类型

Null数据类型只包含唯一的值即null；在初始化变量值时候，常将不需要设置值得时候显性的设置为null；

需要注意的一点是，typeof null的数据类型显示object；null是一个空对象指针，所以显示为一个object；
实际上undefined值是派生自null，ECMA-262规定两者相等操作符为true；null和undefined虽然==true；当两者用法完全不同。

boolean类型

boolean类型即两个字面量：true和false；
这两个值与数字值是不一样的true不一等等于1；false也不一定等于0；

Boolean类型区分大小写，True，和False都不是布尔值，是标识符；
boolean类型的转换：任何一个数据类型的值调用Boolean()函数的时候，总会返回一个Boolean值。

在流程控制语句中会自动执行相应的Boolean转换规则。

Number类型

二进制、十进制、十六进制、八进制等数字类型。八进制的第一位必须为0，然后八进制的序列是0-7；如果数字学列超出了范围，那么前导0将被忽略，当做十进制解析。十六进制字面值前两位必须为0x，后面跟字面0-9、a-f的。其中a-f可以大写也可以小写。

浮点数值

javascript中并没有规定特殊的浮点数类型。浮点数值内存空间为整数数值的两倍

如果小数点后为0，那么将会转换为整数。

对于极大和极小的数值，可以采用科学计数法e表示浮点数。表示方法为前面一个数值，中间为小写或大写的e，后面是10的幂指数。

浮点数计算的时候不准确性：0.1+0.2的值为0.30000000000000004，这个小小的误差会到导致无法测试特定的浮点数值。

数值范围

数值有一定的大小范围，多数浏览器中这个是是5e-324；大多数浏览器中Number.MAX_VALUE和Number.MIN_VALUE保存了最大和最小的数值。
要想确定一个数值是否在-Infinity和Infinity中间，可以使用isFinite()函数。这个函数在参数在最小值和最大值之间会返回true；

###NaN
NaN非数值。这个数值表示在本来要返回一个数值的操作未返回的情况。NaN不等于任何数值。NaN也不等于本身。任何涉及NaN操作都返回NaN；任何数除以0都返回NaN。

isNaN()函数，函数接受一个参数，可以是任何类型，函数帮我们确定参数是否是“不是数字”，不是数字返回true。
isNan()函数也适用对象。当一个对象作为参数的时候，会先调用对象的valueOf()，然后确认其参数是否可以转换为数值，如果不能则调用toString()方法进行转换。

数值转换

数值转换的基本规则是：
a、如果是Boolean值，true和false分别转换为1和0.
b、null和undefined返回0。
c、如果传入的是数值，则返回这个数值。
d、如果是字符串则：
1.字符串只包含数字，如“123”则返回整数。
2.字符串中含浮点格式“1.1”，返回浮点数值。
3.包含十六进制，“ox”则转换为十进制整数值。
4.空字符串转化为0。
5.其余的则转换为NAN。
数值转化中有三个基本函数。Number()、parseInt()、pareseFloat(),分别为转换任何数据类型、转换为整数和浮点数据累心。常见制定第二个参数类型为10是很有必要的。

String类型

组成

String类型是由多个16位的Unicode组成。

转义字符

常见的\n,\b,\r,\等。换行、空格、回车、斜杠等。
字符串可以由str.length获得其长度。当字符串中包含双字节字符，length可能不会反悔准确的字符数目。

##字符串拼接
字符串一旦创建就无法改变。所以当需要改变字符串值得时候需要重新赋值改变。

##转换为字符串
toString()方法可以将每一个值转换为相应的字符串表现。toString以十进制返回数值的字符串表示。当传入2/8/10/16时能够以相应的进制表示字符串。

Object类型

Object类型是一组数据和功能的集合。

每个Object都有下列属性和方法。
1、constructor–构造函数，保存着当前对象的函数。
2、hasOwnProperty–用于检查给定的属性在当前实例中是否存在。

6、valueOf()

面向对象的程序设计

我们经常看到用一个元素模拟的三角形，相信你也已经知道了其中的技术方法。简单来说就是利用

的方法来模拟三角行。

那么我们要模拟一个具体尺寸的三角形时候，这个三角形是怎么算出来的呢？

大边框产生梯形

我们看下面的样式

.box{
    width: 200px;
    height: 200px;
    background: #efefef;
    border:20px solid #ddd;
    border-top:20px solid #000;
    text-align:center;
    line-height: 200px;
}

三角形尺寸的计算

我们知道border为20px；那么他的斜边就为20√2 px；
当一个元素的宽和高都为0的时候，那么三角形的斜边就为相邻边宽度平方和的开根号。

那么当我们设定，上右下左的border宽度分别为 80px 120px 200px 60px，他构造出来的三角形是什么样子的呢？
利用三角形的勾股定理我们能够简单的计算出三角行的形状。这里就不计算了，我们因此能够得到不同角度的三角形，或者线条。

简而言之，就是上下边框的宽度等于左右三角形的底边，左右三角形的宽度等于上下三角形的底边，三角形的高度为各自边框的高度。
