title:sass语法

文件后缀名

sass有.sass和.scss两种后缀文件名。.sass是不使用大括号和分号。.scss是平常使用css文件格式。常规使用.scss文件格式，以避免出现sass格式错误而出现的编译错误。

文件导入

sass的文件导入采用@import '文件名'的方式，在此您可以不添加后缀名。sass导入后会生成一个文件，若以@import 'reset.css'的方式导入，择会以css方式导入，并不会生成一个文件。

变量

sass变量有常规变量采用$开头，后面紧跟变量名，而变量值和变量名之间添加冒号（:）分隔开，如果后面添加!idefault则为默认值。

默认变量

sass的默认变量仅需要在值后面加上 !default 即可。

sass的默认变量一般用来设置默认变量，然后根据需求来覆盖、

变量类型

普通变量

定义之后可以在任何位置使用。

特殊变量使用时需要 #{$variable} 使用

在变量作为属性或在特殊情况下等则必须要以 #{$variables}形式使用。

多值变量

多值变量分为list和map类型。简单来说list类型有点像js中的数组，而map类型有点像js中的对象。

list

list数据可通过空格，逗号或小括号分隔多个值，可用nth($var,$index)取值。list数据操作还有很多函数。参见[sass Functions][http://sass-lang.com/documentation/Sass/Script/Functions.html]

map

map数据以key和value成对出现，其中value又可以是list。格式为：$map:(key1:value,key2:value2,key3:value3)。可以通过map-get($map,$key)取值。map数据还有更多函数具体参见[sass function][http://sass-lang.com/documentation/Sass/Script/Functions.html]

嵌套

&表示本身

选择器嵌套

属性嵌套

###@at-root跳出嵌套

@at-root 和&一起可以在交换选择器嵌套

@at-root只能跳出选择器嵌套。不能跳出@medio或@support，需要跳出@media和@support时使用@at-root(without:media)。常用@media，@support兼容性有问题。

###mixin 混合用$mixin:定义。@include 引用混合。

继承 @extend

@extend 样式块引用。

%定义站位选择器

这种选择器的优势是，在你不调用的时候不会有css文件。这样避免了定义大量的基础样式，却又有很大一部分没有使用的情况
可以定义基础样式，当样式内容引用了基础样式的时候sass才会加载渲染

函数@function

主要使用颜色函数，加深和减淡。

###运算
sass具有运算的特性，可以对数值的Value（如：数字、颜色、变量等）进行加减乘除四则运算。运算符前后需要添加空格进行运算，不然会出错。

条件判断及循环

@if

@if可以单独使用，也可以结合@else多条件使用

三目运算

语法为：if($condition,@if_true,@if_false)。三个参数分别表示条件，条件为真的值，和条件为假的值。

$animal-list:puma,sea-slug,egret,salmander;
@each $animal in $animal-list{
.#{$animal}-icon{
background-image:url(‘/images/#{animal}.png’);
}
}
//css style
.puma-icon{
background-image:url(‘/images/puma.png’);
}
.sea-slug{
background-image:url(‘/images/sea-slug.png’);
}
.egret{
background-image:url(‘/images/egret.png’);
}
.salmander{
background-image:url(‘/images/salmander.png’);
}
